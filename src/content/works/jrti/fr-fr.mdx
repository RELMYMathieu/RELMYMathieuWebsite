Un mod pour [Kerbal Space Program](https://www.kerbalspaceprogram.com/) qui diffuse les flux des caméras **HullCam VDS** sur une page web locale, pour regarder les caméras embarquées de sa fusée depuis un navigateur pendant qu'on joue.

Successeur spirituel de **OfCourseIStillLoveYou (OCISLY)** qui est un mod similaire, mais malheureusement non mis à jour. L'objectif est de pousser plus loin la visualisation externe des caméras dans KSP, en explorant jusqu'où on peut emmener le jeu avec ce genre de setup.
Un des points forts est que ce mod s'intègre dans le pipeline de rendu de KSP pour diffuser les flux des caméras avec le support de tous les mods visuels les plus cool du moment, un support codé à la main, par ma part !

Il s'agit pour l'instant d'une version publique expérimentale / proof of concept. Le projet est versionné en 1.0.0 parce qu'il est conçu pour rester itératif plutôt que d'attendre un état "parfaitement fini" imaginaire.
