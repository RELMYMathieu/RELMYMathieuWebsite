Personal site & maybe somewhat of a "portfolio". Built with Astro — static, fast, not too much overhead. The monospace aesthetic and terminal nav came naturally. It's my little living thing.

Quick demo — I can add **markdown content**, images, and more here! (thanks MDX!)

```typescript
const greeting = "Hello from MDX!";
```

This content gets rendered inside the work window (the one you're looking at right now!), which gives me the freedom to write rich stuff with images, code blocks, and even custom components.
It's honestly just really cool, and multilingual on top of that thanks to how simple Astro makes it.
Feel free to peek at the source on GitHub — it's fully open source! It's a living project that I keep tinkering with, adding things here and there, partly because I enjoy working on it.
The site is also available in French via the navigation bar, so switch between languages if you want to see both versions!
(I haven't been super consistent with the translations — it's a personal project, so I do what I want, haha.)
